구현한 내용:
바닥위에 로봇 4개가 자유롭게 움직인다
각 로봇은 360도 모든 방향으로 움직일 수 있다
가장자리 도달 시 방향을 바꾼다.

카메라 시점 바꾸기 (1인칭, 3인칭)
1, 3 으로 시점 변환
3인칭 시 검은색 큐브로 카메라 표현
3인칭은 1인칭시점 기준의 약간 뒤에서 바라보는 형식으로 구현

w a s d 누르고 있으면 그 방향으로 계속 이동하고, 떼면 이동 멈춘다
y, Y  카메라 기준 양, 음 방향 회전
c, C 중점기준 양, 음 방향 회전

로봇의 탐지범위 내 카메라가 있으면 카메라에 빠르게 다가온다

탐지된 상태에도 카메라가 탐지 범위 밖으로 벗어나면 다시 자유롭게 움직인다

로봇이 카메라와 충돌 시 로봇은 탐지범위와 상관없이 항상 카메라를 쫓아온다


3인칭 시 카메라의 위치에 검은 큐브가 있다

중앙에 장애물이 있고, 장애물에 충돌 하면 충돌한 만큼 밀어내고 움직일 좌표를 갱신한다
